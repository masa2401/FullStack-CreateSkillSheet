# スキルシート作成サイト　概要

---

## 要件

- グラフィカルなスキルシートサイト
- デザイナー/プログラマー関わらず使える
- 会社全体で使う（!?）
- フロントエンドのみで実装
- 質問に回答するだけの簡単な操作感
- 印刷対応（PDF）

## ページ構成（案）

- Top ページ<br>
  ↓
- 質問ページ<br>
  ↓
- 結果ページ（印刷可）

## 技術的視点

- 見た目に関しては<u>Bootstrap</u>で体裁を整える予定
- 学習するフレームワークで構成するとの課題なので、<u>Vue</u>でページを構成したい
- フォーム機能を利用して、入力データを受け取る
- <span style="opacity: 0.4">（入力データ次第で質問内容を変更させる？）</span>
- アルゴリズムに受け取ったデータを入れて、結果を出力（別クラスにした方がいいかも？）
- 印刷対応のサイトデザインについては要リサーチ

## ページ内容

### TOP ページ

- サイト内容説明
- 表示名入力受取 → <u>次ページ以降で表示名利用</u>

### 質問ページ

- フォームの入力機能を使い、１～５の選択式の質問をいくつか並べる

### 結果ページ

- 印刷対応出来る形式で出力
- 回答一覧の表示（いらないかも）

## 検討が必要な内容

- デザイン側の質問内容項目
  - → デザインに関するスキル知識が無さすぎるため
- スキルレベルに関する詳細な記述欄の有無
  - → 利用者の入力量が増える懸念。しかし、詳細な記述があった方がきちんとした情報が伝わる
- デザイナー/プログラマー関わらず使える設計は OK だが、結果出力画面も同一にする必要はあるかどうか
  - → 条件分岐によりふるい分けして結果出力時に独立した形で出力する？
- JavaScript でアルゴリズムの大半を構成する事になりそうなので、クラス構造について考える必要がある
- <span style="opacity: 0.6">レーダーチャート作る
  - → 視覚的に得意分野が分かりやすくなるかも？</span>

## 学習が必要な内容

- Vue の使い方全般
- Vue+Bootstrap 環境構築に関して
- PDF に対応したサイトデザインに関しての学習
